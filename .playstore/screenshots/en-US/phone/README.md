# Phone Screenshots (en-US)

Place 2-8 phone screenshots here for the English (US) locale.

## Requirements

- **Quantity**: Minimum 2, maximum 8 screenshots
- **Format**: PNG or JPG
- **Dimensions**:
  - Minimum: 320px (shortest side)
  - Maximum: 3840px (longest side)
- **Aspect Ratio**: Common phone ratios (16:9, 9:16, etc.)
- **File Size**: Keep under 8MB per screenshot

## Naming Convention

Screenshots will be uploaded in alphabetical order. Use numbered names for consistent ordering:

```
1.png
2.png
3.png
4.png
...
```

Or descriptive names that sort properly:

```
01-home-screen.png
02-feature-one.png
03-feature-two.png
04-settings.png
...
```

## Tips

- Show your app's key features
- Use actual app content, not placeholder text
- Ensure text is readable
- Consider adding text overlays to highlight features
- First 2-3 screenshots are most important (shown in search results)
- Use consistent device frames (optional but professional)

## Tools for Screenshots

- **Fastlane Frameit** - Add device frames
- **Screenshot Framer** - Online tool for frames
- **App Mockup** - Create professional screenshots
- Your device's screenshot feature
