# Tablet Screenshots (en-US)

Place 2-8 tablet screenshots here for the English (US) locale (optional).

## Requirements

Same as phone screenshots:
- **Quantity**: Minimum 2, maximum 8 screenshots
- **Format**: PNG or JPG
- **Dimensions**: Min 320px, max 3840px
- **File Size**: Keep under 8MB per screenshot

## When to Include Tablet Screenshots

Include tablet screenshots if:
- Your app has a tablet-optimized UI
- Your app uses tablet-specific features
- You want to showcase the app on larger screens

## Naming

Use numbered names for ordering:
```
1.png
2.png
3.png
```

Replace this README with actual screenshots if your app supports tablets.
