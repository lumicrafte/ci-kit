# Graphics Assets

Place your Play Store graphics in this directory:

## Required Files

### icon.png
- **Size**: 512x512 pixels
- **Format**: PNG (32-bit PNG with alpha)
- **Purpose**: App icon displayed in Play Store

### feature-graphic.png
- **Size**: 1024x500 pixels
- **Format**: PNG or JPG
- **Purpose**: Large banner displayed at the top of your Play Store listing

## Optional Files

### promo-graphic.png
- **Size**: 180x120 pixels
- **Format**: PNG or JPG
- **Purpose**: Promotional graphic for featuring opportunities

## Tips

- Use high-quality images
- Avoid text in graphics (except icon) as they should work across all locales
- Feature graphic should be visually appealing and represent your app
- Keep file sizes reasonable (under 1MB recommended)
- Use PNG for images with transparency, JPG for photos

## Example Tools

Create graphics with:
- **Figma** - https://figma.com
- **Canva** - https://canva.com
- **Adobe Photoshop**
- **GIMP** (free)
- **Inkscape** (free, vector graphics)
